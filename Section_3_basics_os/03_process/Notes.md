=>   What is process ?

          Any program/app/application which takes memory, cpu time is called process.

          Process is a unit of execution.

          Process executed by a processor.


=>   States of a process :

       (1)  Ready state     =>  Ready for execution.
       (2)  Waiting state
       (3)  Running state
       (4)  Terminated state
       (5)  Sleeping state

=>   Who creates process?
   
     The kernel is responsible for creating processes.
     The Process Manager inside the kernel handles process creation, resource allocation, and scheduling.
     Context switching is managed by the scheduler, which is also part of the kernel.
